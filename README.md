# test.postgresql

## Setup

1. Run `cargo build`
2. Run `cp .env.example .env`
3. Run `docker compose up`
4. Run `cargo run --bin execute <filename in sql/>`

## Pgadmin

Open [http://localhost:5433](http://localhost:5433) for pgadmin. Login with data specified in .env PGADMIN_DEFAULT\_
